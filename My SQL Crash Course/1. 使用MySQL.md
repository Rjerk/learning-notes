# 1. 使用 MySQL

显示所有能用的 show 命令
```
help show;
```

查看拥有数据库
```
show databases;
```

使用数据库 course
```
use course; 
```

查看表
```
show tables;
```

显示表列
```
show columns from customers;
```

显示服务器状态信息
```
show status;
```

实现授权用户的安全权限
```
show grants;
```

显示服务器错误或警告信息
```
show errors;
show warnings;
```
